# Real-World Engineering Skills

Production-grade cognitive modules for AI coding agents.

These skills encode **senior-level engineering judgment**, **policy enforcement**, **systems thinking**, and **risk-driven decision making** directly into your AI collaborator — moving beyond “vibe coding” into disciplined, verifiable, real-world software engineering.

Built as structured `.claude/skills/` modules with **Activation Triggers**, **Mental Models**, **Thinking Rules**, **Policy Enforcement**, and **Verification Checklists**. All tagged **[USER-GROUNDED]**.

### Quick Start

1. **Install skills**: Copy `.claude/skills/` to your Claude skills directory (`~/.claude/skills/` or project `.claude/skills/`).  
2. **Activate**: Use natural language or specific **Activation Triggers** from each skill’s SKILL.md.  
3. **Compose**: Chain skills together for powerful multi-policy workflows.

### Usage Instructions

These skills work best with Claude Code / Claude Projects. They are portable to other advanced coding agents with light adaptation.
